# Persona-to-Model Assignment Rationale Pack
## Purpose
This file explains the logic behind assigning the **primary**, **secondary**, and **support** decision models for **all 10 personas** across the full **4×4 matrix**.
It follows the fixed matrix architecture from the decision framework spec:
- **emotion** determines the dominant reasoning mode
- **perspective** determines what kind of consequence matters
- **persona** changes the weighting, visible data, and wording inside the cell
So the stack is not assigned by persona alone. The stack is assigned by **cell**, then shaped by persona.
